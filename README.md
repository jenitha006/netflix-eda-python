
# Netflix Content Analysis — EDA

## Overview
Exploratory data analysis of 8,000+ Netflix titles using Python.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

## Dataset
Netflix Movies and TV Shows — Kaggle

## Key Findings
- Netflix has significantly more Movies than TV Shows
- Content additions grew explosively between 2015 and 2019
- US leads content production followed by India and Japan
- Netflix primarily targets mature audiences (TV-MA most common)
- International Movies, Dramas and Comedies are top 3 genres
- Movies continue to dominate over TV Shows with a widening gap
